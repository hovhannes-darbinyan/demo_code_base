## Описание суть папок
``` 
 api - тут находится клиенты к бекенду 
 components -  тут находится компоненты катроые данные получают через пропс.
 containers -  тут находится компоненты катроые будут брать данные из стора не имет значение redux либо другой 
 суть в том чтоб добавить ирархию для взятия данных чтоб не было лишного рендеринга.
 store - данном папке находятся функционал стора данном случии redux
 constants - это топ левел константов
 controllers - тут будет лежать бизнес логика вчистом js чтоб интерпертационм левеле был больше перформанса
 hooks - тут катом хуки 
 pages - компоненты для канкретного странииц 
```
